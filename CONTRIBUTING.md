# Contributing to Awesome AgentOS

Contributions are welcome! If you have a repository that fits the "AgentOS" theme, please follow these steps:

1.  **Search the list** to ensure the repository isn't already included.
2.  **Check the repository quality**: It should be open-source, active, and relevant to AI agents or agentic infrastructure.
3.  **Create a Pull Request**:
    *   Add the repository to the most appropriate category.
    *   Use the format: `[Author/RepoName](URL) - Concise one-sentence description in English.`
    *   Keep descriptions objective and clear.
4.  **Wait for review**: I will review your PR as soon as possible.

Thank you for helping make this list awesome!
